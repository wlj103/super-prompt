# 更新日志

## v1.0 (2026-07-23)

### 首次发布

- **七道闸门**：G0 意图澄清 → G1 领域搜索 → G2 反歧义锁定 → G3 结构化草稿 → G4 对抗测试 → G5 ELO 排名 → G6 交付
- **18 条反歧义校验清单**：系统锁定每条歧义路径
- **ELO 排名系统**：K=32，基线 1200，源自 gpt-prompt-engineer（MIT）
- **输出复杂度**：轻/中/重三级，按项目范围自动判定
- **输入模式**：标准/审查/重型，与输出复杂度正交
- **预搜索**：每问必搜，搜索结果转化为选项注入
- **分层引导**：每轮 ≤4 问，从最简单到最深
- **速查卡**：一页提示词摘要，日常使用
- **维护义务**：降级处理、版本追踪、更新触发
- **四平台对比**：与 SkillOpt、prompt-optimizer、gpt-prompt-engineer 对比
- **中英双语**：完整双语 SKILL.md

### 设计原则
1. 预搜索，不猜测
2. 分层引导，不审问
3. 反歧义，不假设
4. 证据驱动，不观点驱动
5. ELO 验证，不人工判断
6. 交付带保险
7. 按需输出，不过度
