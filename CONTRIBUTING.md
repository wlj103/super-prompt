# Contributing to Super-Prompt

Thanks for your interest in contributing!

## How to Contribute

### Found a Bug or Have a Feature Idea?

Open an [Issue](https://github.com/wlj103/super-prompt/issues) with:
- What you expected to happen
- What actually happened
- Steps to reproduce

### Want to Add Code?

1. Fork the repo
2. Create a feature branch (`git checkout -b feature/amazing-feature`)
3. Commit your changes (`git commit -m 'Add amazing feature'`)
4. Push to the branch (`git push origin feature/amazing-feature`)
5. Open a Pull Request

### What We're Looking For

- **New anti-ambiguity patterns** — discovered in practice, with examples
- **New prompt modes** — scenarios where the current gates don't cover well
- **ELO improvements** — better ranking algorithms, calibration methods
- **Documentation** — translations, clearer examples, real-world case studies

## Standards

Super-Prompt follows the [Agent Skills](https://agentskills.io) open standard. All contributions should maintain compatibility with the skill format.
