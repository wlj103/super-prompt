# 参与贡献 Super-Prompt

感谢你的贡献兴趣！

## 如何贡献

### 发现 Bug 或有功能想法？

提交 [Issue](https://github.com/wlj103/super-prompt/issues)，附上：
- 你期望发生什么
- 实际发生了什么
- 复现步骤

### 想贡献代码？

1. Fork 仓库
2. 创建功能分支 (`git checkout -b feature/amazing-feature`)
3. 提交更改 (`git commit -m 'Add amazing feature'`)
4. 推送到分支 (`git push origin feature/amazing-feature`)
5. 提交 Pull Request

### 我们寻找什么

- **新的反歧义模式** — 实践中发现，附实例
- **新的提示词模式** — 当前闸门无法很好覆盖的场景
- **ELO 改进** — 更好的排名算法、校准方法
- **文档** — 翻译、更清晰的示例、真实案例研究

## 标准

Super-Prompt 遵循 [Agent Skills](https://agentskills.io) 开放标准。所有贡献应保持与 skill 格式的兼容性。

## 许可证

通过贡献，你同意你的贡献将按 MIT 许可证授权。
